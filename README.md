## Dev Setup

``` bash
# install dependencies
$ npm ci

# serve with hot reload at localhost:3000
$ npm run dev
```

## Production Setup

```bash
# install dependencies
$ npm ci

# build for production and launch server
$ npm run build
$ npm start
```
